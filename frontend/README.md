# Frontend

Minimal React Router app scaffolded for this project with:

- Tailwind CSS v4 wired through Vite
- Shared `app/components` folder
- Top-level routes plus nested dashboard routes
- A simplified file tree without Docker or starter welcome assets

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Main folders

```text
app/
  components/
  routes/
  app.css
  root.tsx
```
